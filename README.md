# Skeleton-ui  
  
## AVAILABLE SCRIPTS  
  
    `start` - run development service with proxy to api  
    `build` - make production build  
    `clean` - clean dist directory  
    `lint`  - run linter  
    `lint:fix` - run linter with fix flag  
  
## REQUIRED ENV VARIABLES  
  
* **GOOGLE_CLIENT_ID**  
* **API_URL**
